## Step 2: Connect Your Custom Domain

### A. In Vercel
1. Go to your project â†’ **Settings** â†’ **Domains**
2. Click **"Add Domain"**
3. Enter: `shareovercoffee.com`
4. Also add: `www.shareovercoffee.com`
5. Vercel will show you DNS records to add

### B. DNS Configuration

Go to your domain registrar (where you bought shareovercoffee.com) and add these DNS records:

**For shareovercoffee.com:**
```
Type: A
Name: @ (or leave blank)
Value: 76.76.21.21
TTL: Auto or 3600
```

**For www.shareovercoffee.com:**
```
Type: CNAME
Name: www
Value: cname.vercel-dns.com
TTL: Auto or 3600
```

### C. Wait for DNS Propagation
- Takes 5-60 minutes (usually ~15 minutes)
- Vercel will auto-issue SSL certificate
- You'll get HTTPS automatically!

## Troubleshooting

### "Site not found" or DNS errors
- DNS not propagated yet (wait 15-60 minutes)
- Check DNS records are correct
- Try: https://dnschecker.org

### OAuth errors
- Verify redirect URIs exactly match
- Check both origins and redirect URIs are added

### Database connection errors
- Verify DATABASE_URL in Vercel env vars
- Check Neon database is running
